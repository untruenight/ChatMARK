## Description (EN)

ChatMARK — Lightweight bookmark tabs for AI chat conversations.

Save, organize, and navigate bookmarks within ChatGPT web
conversations — without leaving the page.

FEATURES
- Create bookmarks by selecting any text in a conversation
- Color-coded tabs displayed on a right-side rail for quick access
- One-click scroll to any bookmarked position
- Drag-and-drop reorder, pin, and expand bookmark tabs
- Inline search to find bookmarks on the current page
- Undo/redo support for bookmark actions

PRIVACY
- All data stored locally on your device
- No data collection, no external servers, no tracking
- Only requires the "storage" permission

HOW TO USE
1. Open any ChatGPT conversation at chatgpt.com
2. Drag to select text, then click the MARK button
3. Your bookmark appears as a tab on the right rail
4. Click any tab to scroll back to that position

COMING SOON

- Gemini support

ChatMARK — AI 채팅 대화를 위한 가벼운 북마크 탭.

페이지를 떠나지 않고 ChatGPT 웹 대화 내에서
북마크를 저장, 정리, 탐색할 수 있습니다.

기능
- 대화 내 텍스트를 선택하여 북마크 생성
- 오른쪽 레일에 색상별 탭으로 빠르게 접근
- 탭 클릭 한 번으로 북마크 위치로 이동
- 드래그 앤 드롭으로 정렬, 고정, 확장 가능
- 현재 페이지 내 북마크 검색
- 북마크 작업 실행 취소/다시 실행 지원

개인정보
- 모든 데이터는 사용자 기기에만 저장됩니다
- 데이터 수집, 외부 서버 전송, 추적 없음
- "storage" 권한만 사용합니다

사용 방법
1. chatgpt.com에서 아무 대화를 엽니다
2. 텍스트를 드래그해서 선택하고 MARK 버튼을 클릭합니다
3. 오른쪽 레일에 북마크 탭이 나타납니다
4. 탭을 클릭하면 해당 위치로 스크롤됩니다
