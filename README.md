# ChatMARK

Lightweight bookmark tabs for ChatGPT web conversations.

## Features

- Create bookmarks by selecting text in conversations
- Color-coded tabs on a right-side rail
- One-click scroll to bookmarked positions
- Drag-and-drop reorder, pin, and expand
- Current-page bookmark search
- Undo/redo support

## Privacy

- All data stored locally on your device
- No data collection, no external servers, no tracking
- Only requires the `storage` permission

## Build

```bash
npm install
npm run build:all
```
